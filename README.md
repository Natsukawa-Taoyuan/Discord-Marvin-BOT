# Discord-Marvin-BOT
一款多功能的 Discord 伺服器自動化戰術助手。整合了強大的檔案辨識系統（支援各廠牌 RAW 攝錄影格式與專業設計專案檔）、動態語音頻道管理、YouTube 數據追蹤以及進階的安全防禦機制。旨在提供管理員全方位的伺服器運維支持。

# Marvin - 專業 Discord 自動化戰術助手

一個基於 `discord.py` 開發的全方位伺服器自動化機器人，旨在提供管理員高效的維運支持。Marvin 整合了強大的檔案辨識系統、動態語音管理與安全防禦機制，是創意團隊與專業社群的理想戰術小幫手。

## 🌟 核心亮點

### 1. 專業級檔案與 RAW 格式辨識
Marvin 具備精準的擴展名分析邏輯，能自動識別並播報多種專業領域檔案，優化團隊溝通效率：
* **專業設計專案檔**：支援 `.psd` (Photoshop)、`.aep` (After Effects)、`.prproj` (Premiere)、`.blend` (Blender)、`.c4d` (Cinema 4D) 等。
* **相機 RAW 原始檔**：精準辨識各廠牌格式，包括 `.arw` (Sony)、`.cr3` (Canon)、`.raf` (Fujifilm)、`.nef` (Nikon)、`.dng` (Universal) 等。
* **程式碼與腳本**：支援 `.py`、`.js`、`.json`、`.cpp`、`.sh` 等多種程式語言辨識。

### 2. 伺服器營運自動化
* **動態語音系統 (Dynamic Voice)**：自動化建立與回收臨時語音房，確保頻道列表永遠保持整潔，並賦予房主管理權限。
* **實時數據面板**：同步 YouTube 頻道訂閱數與伺服器總人數至語音頻道名稱，實現可視化營運監控。
* **自動化歡迎系統**：提供格式化的歡迎卡片，引導新成員領取身分組並閱讀規範。

### 3. 進階安全防禦機制
* **防洗版與標註攔截**：偵測異常頻率的標註與違禁詞彙，自動執行攔截並記錄至系統日誌。
* **惡意連結掃描**：內建連結連線測試工具，協助成員在開啟未知網址前進行預檢。
* **系統日誌系統**：詳盡記錄訊息編輯、刪除及成員出入紀錄，確保社群活動透明。

## 🛠️ 指令系統概覽

### 管理員專用
* `/toggle`：隨時切換伺服器各項模組開關。
* `/role_spawn`：生成互動式按鈕面板供成員選取身分組。
* `/spawn_verify`：建立一鍵式入群驗證面板。
* `/announce`：發布帶有專業排版與圖片的正式公告。

### 實用工具
* `/check_link`：檢查目標網站的回應速度與存活狀態。
* `/join` / `/leave`：管理輔助語音播報功能的連線狀態。
* `/ping`：測試機器人當前的連線延遲。
